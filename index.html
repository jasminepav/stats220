My meme

My meme is a new version of the classic Theyre the same picture meme from The Office. Since I have only recently finished watching the show, I get more enjoyment from seeing memes from it, and I was keen to use this particular one.

When I opened the image on RStudio for the first time, I immediately wondered how I would figure out the location coordinates for placing my captions on the blank pictures. It was indeed a lot of trial and error to move them into the right spots.

Here is my The Office meme

library(magick)
# Plan: Crop office image into halves, will show use of c() and append by combining these two. Make 2 captions for pictures, show use of image blank and annotate

top <- image_scale(image_read("office_top.jpeg"), "500")
bottom <- image_scale(image_read("office_bottom.jpeg"), "500")

# Make lefthand picture
LH <- image_blank(width = "120", height = "100", color = "#f2f2f2") %>%
  image_annotate(text = "Figuring out\nthe sizes and\nlocations of\nthese boxes", color = "#000000", size = 20, gravity = "center")
LH_10 <- image_rotate(LH, 10)
with_LH <- image_composite(top, LH_10, offset = "+70+40")

# Make righthand picture
RH <- image_blank(width = "180", height = "80", color = "#f2f2f2") %>%
  image_annotate(text = "Many trial and error,\nmuch confusion", color = "#000000", size = 20, gravity = "center")
RH_10 <- image_rotate(RH, 10)
with_RH <- image_composite(with_LH, RH_10, offset = "+300+90")

# Stack top and bottom
meme <- c(with_RH, bottom) %>%
  image_append(stack = T)
meme

# Save
image_write(meme, "my_meme.png")

My animated GIF

My GIF is a parody of the Scottish folk song My Bonnie: My Bunny. During lab 1A, on the pixabay.com site, I found an image of a rabbit looking wistfully out at the ocean which inspired this GIF.

Here is my GIF

library(magick)
# Plan: "My Bonnie" parody

# Read and scale
bunny_1 <- image_scale(image_read("bunny_1.jpg"), "300")
bunny_2 <- image_scale(image_read("bunny_2.jpg"), "300")
bunny_3 <- image_scale(image_read("bunny_3.jpg"), "300")
ocean <- image_scale(image_read("ocean_1.jpg"), "300")

# Make frames with lyrics
frame1 <- bunny_1 %>%
  image_annotate(text = "My bunny lives\nover the ocean...", color = "#ffffff", size = 25, gravity = "northeast")

frame2 <- bunny_2 %>%
  image_annotate(text = "My bunny lives\nover the sea...", color = "#ffffff", size = 25, gravity = "northwest")

frame3 <- ocean %>%
  image_annotate(text = "My bunny lives\nover the ocean...", color = "#ffffff", size = 25, gravity = "northeast")

frame4 <- bunny_3 %>%
  image_annotate(text = "Please bring back\nmy bunny to me!", color = "#ffffff", size = 25, gravity = "northeast")

# Frames vector, animate
frames <- c(frame1, frame2, frame3, frame4)
animation <- image_animate(frames, fps = 0.5)
animation

# Save
image_write(animation, "my_animation.gif")
